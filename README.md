UX Research Project: Interface Design & Cognitive Load

This repository documents an end-to-end UX research project conducted as part of my BSc (Hons) dissertation. The study evaluates how different interface designs influence usability, cognitive load, and user behaviour in an interactive system.

Methods used:

- Literature review

- Usability testing

- Surveys & interviews

- Thematic analysis

- Basic statistical analysis

Key outcomes:

- Identified usability friction points

- Compared user behaviour across interface types

- Produced actionable design recommendations
